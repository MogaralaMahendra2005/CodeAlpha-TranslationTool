# 🌐 Language Translation Tool

A beautiful web-based translation tool supporting 12+ languages.

## Features
- 🌍 Translate between 12+ languages
- 🔄 Swap languages instantly
- 📋 Copy translated text
- 🔊 Text-to-speech
- 📱 Responsive design

## Installation

```bash
pip install -r requirements.txt
python app.py
